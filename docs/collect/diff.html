diff.py¶

collect.diff.get_diff(root, url=None, begin_date=None)[ソース]¶

開始日以降のdiffをとる

パラメータ:

  • root (str) -- プロジェクトのrootへのpath

  • url (str) -- diffをとりたいプロジェクトへのurl.引数がない場合,"./tmp/origin"にあるプロジェクトに対してdiffを取る

  • begin_date (str) -- 測定の開始日.引数がない場合,2000-1-1になる
